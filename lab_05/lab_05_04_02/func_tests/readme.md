# Тесты для лабораторной работы №05_04_02

## Входные данные
Бинарный файл с базой данных о товарах

## Выходные данные
Отсортированная база данных
Товары, наименование которых заканчивается на заданную подстроку

## Позитивные тесты
Сортировки:
1. Товары отсортированы
2. Товары отсортированы в обратном порядке
3. Товары расположены случайно
4. Все товары одинаковые

Поиск по подстроке:
5. Все наименования товаров заканчиваются на заданную подстроку
6. Одно наименование товара заканчивается на заданную подстроку

## Негативные тесты
1. Неверные аргументы командной строки
2. Нет наименований, заканчивающихся на подстроку
