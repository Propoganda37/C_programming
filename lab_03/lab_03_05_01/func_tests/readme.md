# Тесты для лабораторной работы №03_05_01

## Входные данные
Целые n, m; целые элементы матрицы

## Выходные данные
Изменённая матрица

## Позитивные тесты:
01 - тест с матрицей из положительных чисел
02 - тест с матрицей из отрицательных чисел
03 - тест с матрицей из положительных и отрицательных чисел (в матрице есть числа с суммой цифр 10 и
     11 для проверки вычисления суммы цифр)
04 - тест с матрицей, полностью состоящей из чисел с суммой цифр больше 10

## Негативные тесты:
01 - тест с нулевым числом строк
02 - тест с нулевым числом столбцов
03 - тест с нулевым числом строк и столбцов
04 - тест с отрицательным числом строк
05 - тест с отрицательным числом столбцов
06 - тест с отрицательным числом строк и столбцов
07 - тест с числом строк больше 10
08 - тест с числом столбцов больше 10
09 - тест с числом строк и столбцов больше 10
10 - тест с символом вместо числа строк
11 - тест с символом вместо числа столбцов
12 - тест с символом вместо числа строк и столбцов
13 - тест с вещественным числом строк
14 - тест с вещественным числом столбцов
15 - тест с вещественным числом строк и столбцов
16 - тест с символом вместо элемента матрицы
17 - тест с вещественным элементом матрицы
18 - тест с матрицей без чисел, у которых сумма цифр больше 10
