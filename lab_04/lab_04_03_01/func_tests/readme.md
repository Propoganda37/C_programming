# Тесты для лабораторной работы №04_03_01

## Входные данные
Cтрока

## Выходные данные
Слова из строки, отличные от последнего слова, в обратном порядке, в которых
оставили только первые вхождения букв

## Позитивные тесты:
В каждом тесте есть знаки препинания " .,;:?!-"
01 - Строка с разными словами, в некоторых есть одинаковые буквы
02 - Строка с разными словами без одинаковых букв
03 - Строка с несколькими словами такими же, как последнее. Во всех словах повторяются буквы

## Негативные тесты:
01 - Пустая строка
02 - Строка из 257 символов
03 - Строка из >257 символов
04 - Строка со словом из >17 символов
05 - Строка со словом из 17 символов
06 - Строка без слов
07 - Строка из одинаковых слов
08 - Строка из одного слова
