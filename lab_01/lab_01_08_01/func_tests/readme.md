# Тесты для лабораторной работы №01_08_01

## Входные данные
Беззнаковые целые byte1, byte2, byte3, byte4

## Выходные данные
Упакованное число в двоичной СС и полученные после распаковки числа

## Позитивные тесты:
01 - числа из интервала (0-255)
02 - крайние значения
03 - числа из интервала (0-255)

## Негативные тесты:
01 - тест с введёнными буквами вместо чисел
02 - тест с числом большим 255
03 - тест с числом меньшим 0
