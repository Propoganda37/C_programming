# Тесты для лабораторной работы 07_01_03

## Входные данные
Файл целых чисел

## Выходные данные
Отсортированные числа с фильтром или без

## Функциональные тесты

### Позитивные тесты
#### Без фильтра
01 - отсортированный массив
02 - обратно отсортированный массив
03 - случайный массив
04 - массив из одинаковых чисел
05 - массив из 1 элемента
#### С фильтром
06 - один отицательный элемент в массиве
07 - несколько отрицательных элементов
08 - один отрицательный элемент последний
09 - нет отрицательных элементов

### Негативные тесты
01 - нет аргументов
02 - 1 аргумент
03 - несуществующий файл
04 - некорректный аргумент фильтрации
05 - пустой файл
06 - символ в файле
07 - вещественное число в файле
08 - после фильтрации массив становится пустой (единственный отрицательный элемент первый в массиве)

## Модульные тесты

### Сортировка
#### Позитивные тесты
01 - отсортированный массив
02 - обратно отсортированный массив
03 - случайный массив
04 - массив из одинаковых чисел
05 - массив из 1 элемента
06 - сортировка вещественных чисел
07 - сортировка символов

### Фильтрация
#### Негативные тесты
01 - вместо начала массива NULL
02 - вместо конца массива NULL
03 - адрес начала массива больше адреса конца
04 - после фильтрации массив становится пустой (единственный отрицательный элемент первый в массиве)
#### Позитивные тесты
05 - один отицательный элемент в массиве
06 - несколько отрицательных элементов
07 - один отрицательный элемент последний
08 - нет отрицательных элементов
