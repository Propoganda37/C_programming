# Тесты для лабораторной работы №04_02_01

## Входные данные
Cтрока

## Выходные данные
Отсортированная строка без повторяющихся слов

## Позитивные тесты:
В каждом тесте есть знаки препинания " .,;:?!-"
01 - Отсортированная по лексикографическому порядку строка
02 - Отсортированная в обратном порядке строка со словом из 16 символов
03 - Случайно отсортированная строка с повторяющимися строками
04 - Строка из одинаковых слов

## Негативные тесты:
01 - Пустая строка
02 - Строка из 257 символов
03 - Строка из >257 символов
04 - Строка со словом из >17 символов
05 - Строка со словом из 17 символов
06 - Строка без слов
